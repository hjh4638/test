padding — Style in class axes.CategoryAxis

차트 시리즈 아이템이 출력되는 곳의 여백을 설정합니다.
기본적으로 칼럼, 바차트 계열은 padding 속성이 0.5 로 설정되어 있습니다.
라인, 영역 차트를 padding=0 으로 설정한 경우 차트 시리즈 아이템이 출력되는 곳의 여백은 존재하지 않습니다.
단위는 실 차트 아이템 하나의 공간이 1 입니다.
