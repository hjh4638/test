Class DropShadowFilterStyles
 
Packagefilters
Classpublic class DropShadowFilter

그림자 필터를 적용합니다. 그림자 필터는 차트의 seriesFilters 속성이나 시리즈의 filters 속성 값으로 할당됩니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <Line2DSeries  yField="Profit" displayName="Profit">
    <filters>
     <DropShadowFilter distance="3" color="0x666666" alpha=".8" />
    </filters>
  </Line2DSeries>
