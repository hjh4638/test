Packagecharts
Classpublic class RadarChart

RadarChart 클래스는 레이더 차트를 생성하는 컨트롤입니다.
할당된 데이터를 바탕으로 각각의 시리즈를 비교 분석할 수 있는 차트를 생성합니다.
레이더 차트의 type은 다음과 같습니다.
  • "polygon" - 데이터의 개수 만큼 다각형 모양의 레이더 차트를 생성하게 됩니다.
  • "circle" - 데이터의 개수와 관계없이 항상 원형 모양으로 레이더 차트를 생성하게 됩니다

    • RadarChart의 series 속성에는 RadarSeries 객체의 배열이 할당되어야 합니다.

    레이아웃은 다음과 같은 형식으로 작성됩니다.

     <rMateChart backgroundColor="0xFFFFEE"  cornerRadius="12" borderStyle="solid">
   <Options>
    <Caption text="Radar Chart"/>
  </Options>
   <RadarChart id="chart1"
              isSeriesOnAxis="false"
              type="polygon"
     paddingTop="20"
     paddingBottom="20"
              showDataTips="true">
     <series>
      ...
      ...
     </series>
   </RadarChart>
 <rMateChart>
