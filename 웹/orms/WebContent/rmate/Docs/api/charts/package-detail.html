Package chartsClasses
 

The charts package contains rMate Chart Control classes for the rMate charting components.


Classes
 ClassDescription
 Area2DChart Area2DChart는 Y축의 시작점부터 해당 데이터 값까지를 영역으로 색칠하여 표현합니다.
 Bar2DChart Bar2DChart는 데이터의 값에 따라 일련의 가로줄로 데이터를 표현합니다.
 Bar3DChart Bar3DChart는 데이터의 값에 따라 일련의 큐브 형태의 가로 막대로 데이터 데이터를 표현합니다.
 Bubble3DChart Bubble3DChart 는 각각의 데이터 포인트에 대해 3개의 값(x, y, radius)으로 데이터를 표현합니다.
 CircularGauge CircularGauge는 0~360 구간의 원형 게이지입니다.
 Column2DChart Column2DChart는 데이터의 값에 따라 일련의 세로줄로 데이터를 표현합니다.
 Column3DChart Column3DChart는 데이터의 값에 따라 일련의 큐브 형태의 막대로 데이터를 표현합니다.
 Combination2DChart Combination2DChart는 여러 차트를 조합해 놓은 형태로 데이터를 표현할 때 유용한 차트입니다.
 Combination3DChart Combination3DChart는 2D, 3D를 포함한 여러 차트를 조합해 놓은 형태로 데이터를 표현할 때 유용한 차트입니다.
 Displayer 히스토리 차트의 선택된 영역을 보여주는 디스플레이어 차트입니다.
 HalfCircularGauge 180~360 구간을 표현하는 반원 게이지입니다.
 HCylinderGauge HCylinder는 가로 방향의 실린더 모양으로 목표치를 설정하고 실적을 나타내는 방법으로 사용하게 됩니다.
 HistoryChart 히스토리 차트는 디스플레이어(Displayer), 네비게이터(Navigator), 셀렉터(Selector)로 구성되어 있습니다.
 HistoryRangeSelector 히스토리 차트의 셀렉터에 해당되는 객체입니다.
 HLinearGauge HLinear는 가로 방향의 막대 모양과 목표치를 나타내는 선으로 구성되어 있습니다.
 ImageChart ImageChart는 데이터의 값을 이미지로 표현합니다.
 Line2DChart Line2DChart 컨트롤은 데이터 포인트와 포인트를 선으로 연결함으로써 데이터를 표현합니다.
 Matrix2DChart Matrix2DChart는 각각의 데이터 표현을 x,y,z로 하게 됩니다.
각 데이터는 수평 축을 따르는 x, 수직 축을 따르는 y, 크기를 결정하는 z로 구분됩니다.
 Navigator 히스토리 차트의 전체 데이터를 표시하고 있는 네비게이터 차트입니다.
 Pie2DChart Pie2DChart 는 데이터를 그라데이션이 들어간 2D 형태의 파이 모양으로 데이터를 표현합니다.
 Pie3DChart Pie3DChart 는 데이터를 3D 형태의 파이 모양으로 데이터를 표현합니다.
 Plot2DChart Plot2DChart 컨트롤은 X축과 Y축 구간의 데이터 포인트를 심볼로 표현합니다.
 RadarChart RadarChart 클래스는 레이더 차트를 생성하는 컨트롤입니다.
 RealTimeChart RealTimeChart 클래스는 데이터를 시간 단위와 사이즈 단위로 보여주는 역할을 하는 차트 컨트롤 입니다.
 VCylinderGauge VCylinder는 세로 방향의 실린더 모양으로 목표치를 설정하고 실적을 나타내는 방법으로 사용하게 됩니다.
 VLinearGauge VLinear는 세로 방향의 막대 모양과 목표치를 나타내는 선으로 구성되어 있습니다.
