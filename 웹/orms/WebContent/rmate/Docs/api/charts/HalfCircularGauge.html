Class HalfCircularGaugeStyles
 
Packagecharts
Classpublic class HalfCircularGauge

180~360 구간을 표현하는 반원 게이지입니다.

레이아웃은 아래와 같이 작성합니다.

  <rMateChart backgroundColor="0xEEEEEE" cornerRadius="12" borderStyle="solid">
  <Options>
   <Caption text="Gauge"/>
  </Options>
  <NumberFormatter id="numFmt" precision="0"/>
  <HalfCircularGauge valueChangeFunction="valueChangeFunc"
     frameType="flat"
     minimumAngle="185" maximumAngle="355"
     minimum="0" maximum="500" value="250"
     interval="100" minorInterval="25"
     formatter="{numFmt}"
     padding="10"
     tickGap="10"
     labelGap="5"
     valueXOffset="0" valueYOffset="-80"
     tickLabelStyleName="tickText"
     valueLabelStyleName="valueLabelStyle"
     editMode="true" liveDragging="true"
     showDataTip="true"
     needleCoverRadius="5"
     needleThickness="10"
     pointThickness="0"
     needleLengthRatio="0.75"
     needlePointStyle="steeple"
     needleBackLengthRatio="0.12"
     tickLabelPlacement="inside"
     showTrackColor="false">
   <frameStroke>
    <Stroke color="0x232323" weight="6"/>
   </frameStroke>
   <tickFill>
    <SolidColor color="0xCCCCCC"/>
   </tickFill>
   <minorTickFill>
    <SolidColor color="0xFF9900"/>
   </minorTickFill>
  </HalfCircularGauge>
  <Style>
   .tickText
   {
    fontFamily:tahoma;
    fontFamily:Myriad;
    fontSize:15;
    color:#FFFFFF;
   }
   .valueLabelStyle
   {
    fontFamily:Myriad;
    fontSize:15;
    paddingTop:4;
    textAlign:center;
   }
  </Style>
  </rMateChart>


Styles
 StyleDescriptionDefined by
  
bottomRadius
Type: Number   Format: Length
반원형 게이지 밑단의 중앙 반지름을 픽셀단위로 지정합니다. frameType 을 rouding 으로 설정 시 밑단 중앙의 반지름이 되며, flat 으로 설정 시 bottomRadius 만큼의 여백을 갖습니다. The default value is 0.		HalfCircularGauge
  
frameType
Type: String
반원형 게이지의 밑단의 중앙을 둥굴게 처리할지 평평하게 처리할지를 나타냅니다. 유효값은 "rounding","flat" 입니다. The default value is 'flat'.		HalfCircularGauge
