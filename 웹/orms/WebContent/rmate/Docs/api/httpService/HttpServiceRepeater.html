Class HttpServiceRepeaterStyles
 
PackagehttpService
Classpublic class HttpServiceRepeater

HttpService 를 상속받아 만든 클래스로 설정한 url 을 interval 간격으로 sending을 합니다. 요청으로 받은 데이터를 target 에 넘겨주는 역할을 합니다. 그렇기 때문에 target 은 반드시 dataProvider 속성을 갖는 클래스여야 합니다. serviceStart() 함수를 호출함으로써 interval 간격으로 url 요청을 시작합니다.

<rmate:HttpServiceRepeater> 는 기본적으로 parent classes 의 모든 properties 를 상속받으며 아래의 properties 를 추가로 갖습니다.
