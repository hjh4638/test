Class VTarget3DGoalSeriesStyles

Classpublic class VTarget3DGoalSeries
InheritanceVTarget3DGoalSeries Column3DSeries

VTarget3DGoalSeries 클래스는 목표 대비 실적 차트 즉, Target3DChart 컨트롤의 목표에 해당되는 데이터 시리즈를 정의합니다. 이 클래스를 Target3DChart 컨트롤의 series 속성의 배열로 할당함으로써 목표 대비 실적 차트를 생성 할 수 있습니다. 정상적인 목표 대비 실적 차트를 생성하기 위해서는 시리즈 배열을 정의 할 때 실적에 해당되는 클래스(VTarget3DResultSeries) 인스턴스를 먼저 정의하고, 다음으로 목표에 해당되는 클래스(VTarget3DGoalSeries) 인스턴스를 정의하여야 합니다. 디폴트 아이템렌더러는 CylinderTargetItemRenderer 클래스입니다. columnWidthRatio 속성의 디폴트 값은 0.7 입니다. fill 스타일의 디폴트 값은 0xCCCCCC 색깔을 갖는 SolidColor 인스턴스입니다.

구체적인 정의 방법은 아래와 같습니다.

   <rmate:Target3DChart>
     <rmate:series>
    실적
    <rmate:VTarget3DResultSeries />
  
    목표 
    <rmate:VTarget3DGoalSeries />
     </rmate:series>
   </rmate:Target3DChart>

<rmate:Target3DChart> 는 기본적으로 parent classes 의 모든 properties 를 상속받습니다.
