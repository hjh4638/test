Classpublic class ImageSeries

ImageChart 컨트롤의 series property에 할당되는 데이터 시리즈 입니다. 이미지 차트를 생성하기 위해 ImageSeries 의 imgSource 속성을 반드시 정의해야 합니다. imgSource 는 ImageSourceItem의 배열 조합을 할당하십시오. imageDisplayType 의 유효값은 아래와 같습니다.
  • "single": 하나의 이미지를 이용하여 데이터를 표현합니다. imgSource 에 ImageSourceItem 배열을 할당할 때 maintainAspectRatio의 true, false 에 따라 다르게 표현합니다. true 인 경우 막대 위에 이미지가 정비율로 표현됩니다. false 인 경우 차등비율로 데이터영역 전체를 표현합니다. imgSource 에 여러 ImageSourceItem 을 삽입한 경우 단 하나의 ImageSourceItem를 제외한 ImgSourceItem은 무시됩니다.
  • "singleRepeat": 하나의 이미지를 이용하여 데이터를 표현합니다. 해당 이미지는 정비율로 표현되며 데이터 영역 전체를 표현할 때 까지 반복하여 출력됩니다. ImageSoureItem 의 maintaAspectRatio 값은 항상 true 입니다. imgSource 에 여러 ImageSourceItem 을 삽입한 경우 단 하나의 ImageSourceItem를 제외한 ImgSourceItem은 무시됩니다.
  • "multiple": 여러 이미지로 데이터를 표현합니다. imgSource 속성에 ImageSourceItem 배열을 할당하여 여러 이미지를 정의하십시오. 각각 이미지에 value 를 지정하여 이미지에 단위를 지정할 수 있습니다. ImageSoureItem 의 maintaAspectRatio 값은 항상 false 입니다. value 가 큰 이미지부터 순서대로 출력하여 value가 작은 이미지가 최상위에 표현됩니다.
디폴트로 ImageItemRenderer 클래스를 itemRenderer 스타일 값으로 사용합니다.

<rmate:ImageSeries> 는 기본적으로 parent classes 의 모든 properties 를 상속받으며 아래의 properties 를 추가로 갖습니다. 

   <rMateChart backgroundColor="0xFFFFFF"  cornerRadius="12" borderStyle="solid">
   <Options>
     <Caption text="Image Chart" />
     <SubCaption text="(2004 ~ 2008)" textAlign="center"/>
    </Options>
    <NumberFormatter id="numFmt" /> 
    <ImageChart id="chart" showDataTips="true" gutterLeft="20" gutterRight="20"  showLabelVertically="true">
     <horizontalAxis>
      <CategoryAxis id="hAxis" categoryField="Year"/>
    </horizontalAxis>
     <verticalAxis>
      <LinearAxis id="vAxis"/>
     </verticalAxis>
    <series>
     <ImageSeries yField="Data1" imageDisplayType="single" styleName="seriesStyle" displayName="Male" formatter="{numFmt}">
      <imgSource>
               <ImageSourceItem maintainAspectRatio="true" url="../Samples/Images/1-1-1.png"/>
               </imgSource>
               <showDataEffect>
                <SeriesSlide duration="1000" direction="up"/>
               </showDataEffect>
     </ImageSeries>
     <ImageSeries yField="Data2" imageDisplayType="single" styleName="seriesStyle2" displayName="Female" formatter="{numFmt}">
      <imgSource>
                <ImageSourceItem maintainAspectRatio="true" url="../Samples/Images/1-1-2.png"/>
              </imgSource>
              <showDataEffect>
            <SeriesSlide duration="1000" direction="up"/>
            </showDataEffect>
    </ImageSeries>
   </series>
   <verticalAxisRenderers>
    <Axis2DRenderer axis="{vAxis}" visible="false" includeInLayout="false"/>
   </verticalAxisRenderers>
  </ImageChart>
  <Style>
   .seriesStyle
   {
    labelPosition:outside;
    fontFamily:Myriad;
    color:0x0066FF;
    fontSize:11;
   }
   .seriesStyle2
   {
    labelPosition:outside;
    fontFamily:Myriad;
    color:0xFF66FF;
    fontSize:11;
   }
  </Style>
  lt;/rMateChart>

columnWidthRatio

표시하는 칼럼(column)의 width 를 카테고리의 width에 대한 비율(범위는 0~1)로 지정합니다. 값이 1인 경우는 영역 전체가 사용되며, 0.6인 경우 칼럼의 사용 가능 영역은 60%입니다.

Type: Object
시리즈에서 현재 차트아이템의 fill 컬러를 리턴하는 함수를 정의 합니다. 만약 이 함수가 정의 되었다면, fill 과 fills 는 fillJsFunction 에서 리턴되는 값 때문에 무시됩니다. 그러나 null 값이 리턴 되었을 경우, fills 또는 fill 이 적용됩니다. 		ImageSeries
  
fills
Type: Array
시리즈에서 각각의 아이템의 fill 컬러를 지정하는 fill 객체 배열을 정의합니다. 만약 시리즈의 모든 아이템에 충분한 fill 컬러를 지정하지 않았다면 지정된 색깔이 반복됩니다. 예를 들어, 시리즈의 아이템이 10개 인 경우 5개의 fill 컬러를 지정하였다면 5개를 칠한 후 다시 5개를 칠하게 됩니다.

CSS에서 fills 설정 한 경우: 

     ImageSeries {
       fills:#CC66FF, #9966CC, #9999CC;
     }

MXML에서 fills 설정 한 경우: 

     <rmate:ImageSeries ... >
      <rmate:fills>
       <mx:SolidColor color="0xCC66FF"/>
       <mx:SolidColor color="0x9966CC"/>
       <mx:SolidColor color="0x9999CC"/>
      </rmate:fills>
     </rmate:ImageSeries>
