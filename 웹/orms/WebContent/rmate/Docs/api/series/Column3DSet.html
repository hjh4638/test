Packageseries
Classpublic class Column3DSet

Column3DSeries 들만을 묶어 Column3DSet 으로 대표되는 시리즈 계열을 만듭니다.
Column3DSet으로 대표되는 복수의 Column3DSeries 는 마치 하나의 시리즈와 같이 작동됩니다.
라인과 칼럼이 함께 있는 컴비네이션 차트를 생성 시 칼럼3D시리즈들을 Column3DSet으로 묶어 정의해야 합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

 <Combination3DChart showDataTips="true">
   <horizontalAxis>
    <CategoryAxis categoryField="Month"/>
      </horizontalAxis>
      <series>
         <Column3DSet type="clustered">
           <series>
      <Column3DSeries yField="Cost" displayName="Cost"/>
      <Column3DSeries yField="Revenue" displayName="Revenue"/>
       </series>
      </Column3DSet>
    <Line2DSeries  yField="Revenue" displayName="Revenue"/>
   </series>
  </Combination3DChart>


Styles
 StyleDescriptionDefined by
  
columnWidthRatio
Type: Number
표시하는 칼럼(column)의 width 를 카테고리의 width에 대한 비율(범위는 0~1)로 지정합니다.
값이 1인 경우는 영역 전체가 사용되며, 0.6인 경우 칼럼의 사용 가능 영역은 60%입니다.

실제 칼럼(column)의 width 는 columnWidthRatio property 와 maxColumnWidth property 를 계산하여 작은 값이 적용됩니다

The default value is 0.65.		Column3DSet

Column3DSet
  
type
Type: String
Column3DSet 으로 Column3DSeries 를 묶은 경우 해당 시리즈의 표현 유형을 나타냅니다.
유효한 값은 아래와 같습니다.
  • "clustered": 해당 시리즈를 옆에 나열하듯 그대로 표현합니다. 이것이 디폴트입니다.
  • "overlaid": 복수의 시리즈들을 덮어씌우듯 표현합니다. 마지막에 지정된 시리즈가 맨 위에 나타납니다.
  • "stacked": 복수의 시리즈들을 쌓아올려 표현합니다. 모든 시리즈는 그 아래에 있는 모든 값의 누적치를 나타냅니다.
  • "100%": 복수의 시리즈들을 쌓아올려 표현하되 세로 영역을 100%로 잡고 출력시킵니다.
The default value is "clustered".		Column3DSet
