Class rMateChartStyles
 
PackageTop Level
Classpublic class rMateChart

rMate 차트 레이아웃 작성 시 루트 노드에 해당됩니다. 생성되는 차트의 배경 및 프레임을 담당하게 됩니다.
차트 전체의 배경색 및 프레임의 스타일, 여백등을 조절할 수 있습니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

 <rMateChart backgroundColor="0xFFFFEE"  cornerRadius="12" borderStyle="solid" paddingRight="10" paddingBottom="5" paddingTop="5" paddingLeft="10">
   <Options>
    <Caption text="Anual Report"/>
   </Options>
   ...
   ...
   ...
 </rMateChart>

backgroundAlpha
Type: Number

배경의 투명도를 지정합니다. 유효 범위는 0~1입니다.

The default value is 1.		rMateChart
  
backgroundColor
Type: uint   Format: Color

배경색을 RGB로 지정합니다.

The default value is 0xFFFFFF.		rMateChart
  
borderColor
Type: uint   Format: Color

차트 외형 테두리색을 RGB로 지정합니다.

The default value is 0xB7BABC.		rMateChart
  
borderStyle
Type: String

차트 외형 테두리 스타일을 지정합니다. 유효값은 "none", "solid", "inset", 그리고 "outset" 입니다.

The default value is "none".		rMateChart
  
borderThickness
Type: Number   Format: Length

차트 외형 테두리 선의 두께를 지정합니다. borderStyle 이 solid 로 설정된 경우에만 유효합니다.

The default value is 1.		rMateChart
  
cornerRadius
Type: Number   Format: Length

차트 외형 프레임의 코너 부분을 둥그스름하게 만듭니다.

The default value is 0.		rMateChart
  
paddingBottom
Type: Number   Format: Length

차트 전체 영역의 하단 여백을 픽셀단위로 지정합니다.

The default value is 15.		rMateChart
  
paddingLeft
Type: Number   Format: Length

차트 전체 영역의 왼쪽 여백을 픽셀단위로 지정합니다.

The default value is 15.		rMateChart
  
paddingRight
Type: Number   Format: Length

차트 전체 영역의 오른쪽 여백을 픽셀단위로 지정합니다.

The default value is 15.		rMateChart
  
paddingTop
Type: Number   Format: Length

차트 전체 영역의 상단 여백을 픽셀단위로 지정합니다.

The default value is 20.		rMateChart
