Class CurrencyFormatterStyles
 
Packageformatters
Classpublic class CurrencyFormatter

CurrencyFormatter는 숫자 포맷을 변경하거나 숫자 앞 또는 뒤에 특수 문자를 삽입할 수 있도록 합니다.
예를 들어 통화 단위 "$" 를 앞에 삽입하거나, "원"을 뒤에 삽입 할 수 있도록 처리할 수 있습니다.

The default value is "left".		CurrencyFormatter
  
currencySymbol
Type: String

숫자 앞 또는 뒤에 추가할 문자를 정의합니다. 보통 통화 단위를 삽입하나 일반적인 스트링 삽입이 가능합니다.

The default value is "$".		CurrencyFormatter
