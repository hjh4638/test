Class DateFormatterStyles
 
Packageformatters
Classpublic class DateFormatter

DateFormatter는 날짜 포맷을 변경하여 보일 수 있도록 합니다.
rMate 차트의 기본 날짜 포맷은 YYYY/MM/DD HH:NN:SS 입니다.
그 외 값은 날짜가 아닌 일반 스트링으로 처리하기 때문에 날짜 포맷으로 삽입하기 위해서는 위 포맷형식을 지킬 필요가 있습니다.


Styles

formatString
Type: String

출력하고자 하는 날짜 포맷을 지정합니다.

 ┌──────────────────────────────────┐
 │ Y │ 연(Year)  │ YY=05,YYYY=2005  │
 │ M │ 월(Month) │ M=7,MM=07,MMM=Jul│
 │ D │ 일(Day)   │ D=4,DD=05        │
 │ H │ 시(Hour)  │ H=1,HH=01        │
 │ N │ 분(Minute)│ N=1,NN=01        │
 │ S │ 초(Second)│ SS=30            │
 └──────────────────────────────────┘
예를 들어 날짜 포맷을 "2009년 3월" 과 같이 표현하고자 하는 경우 formatStirng="YYYY년 M월" 이 됩니다. 		DateFormatter
