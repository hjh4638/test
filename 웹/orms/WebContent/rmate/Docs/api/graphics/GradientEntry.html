Class GradientEntryStyles
 
Packagegraphics
Classpublic class GradientEntry

GradientEntry 는 그라데이션 채우기 요소로 LinearGradient 와 RadialGradient 의 entries 속성 값으로 할당됩니다.
GradientEntry 가 정의된 수로 그라데이션 처리가 이루어 집니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <fill>
    <RadialGradient angle="45" focalPointRatio="-0.5">
     <entries>
      <GradientEntry color="0xC5C551" ratio="0.00" alpha="0.5"/>
      <GradientEntry color="0xFEFE24" ratio="0.33" alpha="0.5"/>
      <GradientEntry color="0xECEC21" ratio="0.66" alpha="0.5"/>
     </entries>
    </RadialGradient>
  </fill>
