Class LinearGradientStyles
 
Packagegraphics
Classpublic class LinearGradient

채우기 색을 단색(SolidColor)가 아닌 Linear 그라디에션 처리를 하여 색을 채우도록 설정합니다.
LinearGradient 의 속성인 entires 에 반드시 GradientEntry 집합을 정의해야 합니다.

레이아웃은 다음과 같은 형식으로 작성됩니다.

  <fill>
    <LinearGradient angle="45">
     <entries>
      <GradientEntry color="0xC5C551" ratio="0.00" alpha="0.5"/>
      <GradientEntry color="0xFEFE24" ratio="0.33" alpha="0.5"/>
      <GradientEntry color="0xECEC21" ratio="0.66" alpha="0.5"/>
     </entries>
    </LinearGradient>
  </fill>


Styles
 StyleDescriptionDefined by
  
angle
Type: Number

그라데이션의 진행 방향을 정의합니다. 기본 값은 0 으로 왼쪽에서 오른쪽으로 진행됩니다.
오른쪽에서 왼쪽으로 진행되게끔 설정하려면 angle은 180 입니다.

The default value is 0.		LinearGradient
  
entries
Type: Object
그라데이션 요소를 정의합니다. 유효 값은 GradientEntry 입니다. 		LinearGradient
  
id
Type: String

id를 설정합니다. id는 사용자에 의해 정의된 임의의 고유값입니다.

설정된 id를 통해 해당 객체에 접근이 가능합니다.
