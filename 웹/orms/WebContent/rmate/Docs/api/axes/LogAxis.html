Class LogAxisStyles
 
Packageaxes
Classpublic class LogAxis

데이터의 수치값이 아주 큰 수 인 경우 Log 로 데이터를 표현할 때 LogAxis 를 사용합니다.
LogAxis 는 10의 승수로 표현됩니다.


Styles
 StyleDescriptionDefined by
  
baseAtZero
Type: Boolean

축의 기본 시작이(base) 0 인지를 나타냅니다. 예를 들어 모든 수치값이 양수라면 minimum 은 0 으로, 모든 수치값이 음수라면 maximum 은 0으로 설정됩니다.

The default value is true.		LogAxis
  
displayName
Type: String

축의 이름을 지정합니다. 설정된 값은 기본적으로 데이터팁(툴팁)에서 보여지게 됩니다.

LogAxis
  
formatter
Type: Object

축 수치 라벨을 NumberFormatter 또는 CurrencyFormatter 로 재설정 하여 출력시키게끔 포매터를 지정합니다.

LogAxis
  
id
Type: String

id를 설정합니다. id는 사용자에 의해 정의된 임의의 고유값입니다.

설정된 id를 통해 해당 객체에 접근이 가능합니다.
LogAxis
  
interval
Type: Number

축 라벨에 표시되는 수치의 사이 간격을 10의 승수로 설정합니다. 예를 들어 2로 설정하면 10의 2승인 100 단위로 interval이 설정됩니다.

LogAxis
  
labelJsFunction
Type: String
축에 표시되는 라벨을 사용자 정의하기 위한 자바스크립트 함수명을 정의합니다.
layout XML 에서 속성을 넣을때 labelJsFunction를 주고,만든 javascript 함수명을 넣어줍니다. 
 예) <horizontalAxis>
        <LogAxis id="hAxis" labelJsFunction="axisLabelFunc" />
    </horizontalAxis>

labelJsFunction 의 signature 는 아래와 같습니다. 

 function_name(value) { ... }
  value : 현재 아이템에 맞는 축의 라벨 값.
LogAxis
  
maximum
Type: Number

축 라벨 수치의 최대값을 명시적으로 설정합니다.

LogAxis
  
minimum
Type: Number

축 라벨 수치의 최소값을 명시적으로 설정합니다.

LogAxis
  
minorInterval
Type: Number

축 라벨에 표시되는 마이너 틱(minor tick)의 수치 간격을 명시적으로 설정합니다.

LogAxis
  
padding
Type: Number

축 최소값과 최대값에 각각에 padding 값 만큼 더한 값을 minimum 과 maximum 으로 설정하도록 추가값을 삽입합니다.

The default value is 0.		LogAxis
  
title
Type: String

축의 제목을 지정합니다. 세로축인 경우 세로로 제목이 출력됩니다.
