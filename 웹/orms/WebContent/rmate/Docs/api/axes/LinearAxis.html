Class LinearAxisStyles
 
Packageaxes
Classpublic class LinearAxis

데이터 중 수치에 해당되는 값을 표현하기 위해 정의하는 축이 LinearAxis 입니다.
보통 칼럼, 라인 차트의 세로축에 정의됩니다.


Styles
 StyleDescriptionDefined by
  
baseAtZero
Type: Boolean

축의 기본 시작이(base) 0 인지를 나타냅니다. 예를 들어 모든 수치값이 양수라면 minimum 은 0 으로, 모든 수치값이 음수라면 maximum 은 0으로 설정됩니다.

The default value is true.		LinearAxis
  
direction
Type: String

축의 방향을 지정합니다. 유효값은 "normal", "inverted" 입니다. "inverted" 로 설정하면 축의 방향이 반대가 됩니다.

The default value is "normal".		LinearAxis
  
displayName
Type: String

축의 이름을 지정합니다. 설정된 값은 기본적으로 데이터팁(툴팁)에서 보여지게 됩니다.

LinearAxis
  
formatter
Type: Object

축 수치 라벨을 NumberFormatter 또는 CurrencyFormatter 로 재설정 하여 출력시키게끔 포매터를 지정합니다.

LinearAxis
  
id
Type: String

id를 설정합니다. id는 사용자에 의해 정의된 임의의 고유값입니다.

설정된 id를 통해 해당 객체에 접근이 가능합니다.
LinearAxis
  
interval
Type: Number

축 라벨에 표시되는 수치의 사이 간격을 명시적으로 설정합니다. 보통 차트에 의해 자동 계산됩니다.

LinearAxis
  
labelJsFunction
Type: String
축에 표시되는 라벨을 사용자 정의하기 위한 자바스크립트 함수명을 정의합니다.
layout XML 에서 속성을 넣을때 labelJsFunction를 주고,만든 javascript 함수명을 넣어줍니다. 
 예) <horizontalAxis>
        <LinearAxis id="hAxis" labelJsFunction="axisLabelFunc" />
    </horizontalAxis>

labelJsFunction 의 signature 는 아래와 같습니다. 

 function_name(value) { ... }
  value : 현재 아이템에 맞는 축의 라벨 값.
LinearAxis
  
maximum
Type: Number

축 라벨 수치의 최대값을 명시적으로 설정합니다.

LinearAxis
  
minimum
Type: Number

축 라벨 수치의 최소값을 명시적으로 설정합니다.

LinearAxis
  
minorInterval
Type: Number

축 라벨에 표시되는 마이너 틱(minor tick)의 수치 간격을 명시적으로 설정합니다. 보통 차트에 의해 자동 계산됩니다.

LinearAxis
  
padding
Type: Number

축 최소값과 최대값에 각각에 padding 값 만큼 더한 값을 minimum 과 maximum 으로 설정하도록 추가값을 삽입합니다.

The default value is 0.		LinearAxis
  
title
Type: String

축의 제목을 지정합니다. 세로축인 경우 세로로 제목이 출력됩니다.
