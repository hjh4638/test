Class CategoryAxisStyles
 
Packageaxes
Classpublic class CategoryAxis

스트링 형태를 정의하는 축이 CategoryAxis입니다.
이는 보통 차트에서 수치화 할 수 없지만 그룹화가 가능한 계열을 위한 것입니다.
예를 들면 경영부, 연구부, 회계부와 같이 수치화 될 수 없지만 일개 그룹으로 묶어 축 필드로 사용하고자 할 때 CategoryAxis를 사용합니다.
즉, 데이터 그대로를 출력하고자 할 경우 사용됩니다. 보통 칼럼, 라인 차트 등의 수평 축에 할당됩니다.

Type: String

카테고리축이 위치한 축의 라벨들에 표시할 데이터 필드를 지정합니다.
예를 들어 horizontalAxis 에 CategoryAxis를 할당하였다면 X축 라벨은 categoryField의 데이터가 출력됩니다.

설정된 id를 통해 해당 객체에 접근이 가능합니다.
CategoryAxis
  
labelJsFunction
Type: String
축에 표시되는 라벨을 사용자 정의하기 위한 자바스크립트 함수명을 정의합니다.
layout XML 에서 속성을 넣을때 labelJsFunction를 주고,만든 javascript 함수명을 넣어줍니다. 
 예) <horizontalAxis>
        <CategoryAxis id="hAxis" categoryField="Month" labelJsFunction="axisLabelFunc" />
    </horizontalAxis>

labelJsFunction 의 signature 는 아래와 같습니다. 

 function_name(value) { ... }
  value : 현재 아이템에 맞는 축의 라벨 값.
CategoryAxis
  
padding
Type: Number

차트 시리즈 아이템이 출력되는 곳의 여백을 설정합니다.
기본적으로 칼럼, 바차트 계열은 padding 속성이 0.5 로 설정되어 있습니다.
라인, 영역 차트를 padding=0 으로 설정한 경우 차트 시리즈 아이템이 출력되는 곳의 여백은 존재하지 않습니다.
단위는 실 차트 아이템 하나의 공간이 1 입니다.

The default value is based on the chart..		CategoryAxis
