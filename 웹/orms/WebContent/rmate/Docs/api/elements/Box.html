Classpublic class Box

Label 또는 Image 등과 같은 객체를 담을 수 있는 엘레멘트입니다.
CanvasElement 에 담긴 객체들은 해당 객체의 x,y 좌표대로 출력이 되는 반면 Box 엘레멘트는 x,y 좌표를 무시하고 정의한 노드 순서대로 출력이됩니다.
