Class SeriesInterpolateStyles
 
Packageeffects
Classpublic class SeriesInterpolate

SeriesInterpolate 는 기존의 데이터에서 새로운 데이터로 변화하는 과정을 애니메이션으로 재생시키는 이펙트입니다.
최초의 값은 0 또는 null 로 간주하여 재생됩니다.
모든 시리즈의 showDataEffect 속성의 값으로 할당하십시오.


Styles
 StyleDescriptionDefined by
  
duration
Type: Number
이펙트가 총 재생 시간을 밀리 세컨드 단위로 지정합니다.
즉, 모든 이펙트는 duration에서 정의한 시간 안에 재생되고 종료됩니다. The default value is 500.		SeriesInterpolate
  
elementOffset
Type: Number
시리즈 내 각 아이템의 이펙트를 지연 시키는 시간을 밀리 세컨드 단위로 지정합니다.
elementOffset 를 0 으로 설정하면 모든 아이템의 이펙트가 동시에 시작해 동시에 끝나게 됩니다.
The default value is 20.		SeriesInterpolate

SeriesInterpolate
  
minimumElementDuration

duration 값이 이 값보다 작은 경우 duration 은 무시됩니다.

The default value is 0.		SeriesInterpolate
  
offset

다수의 시리즈를 정의했을 때, 시간 차이를 내며 이펙트를 재생하고자 하는 경우 이 속성을 사용합니다.

The default value is 0.		SeriesInterpolate
