Class SeriesSlideStyles
 
Packageeffects
Classpublic class SeriesSlide

SeriesSlide 는 좌우, 또는 우좌, 상하, 하상 등으로 슬라이딩하는 효과를 재생하는 이펙트입니다.
기존 데이터와 무관하게 항상 설정된 방향(direction)으로 재생됩니다.
모든 시리즈의 showDataEffect 속성의 값으로 할당하십시오.


Styles
 StyleDescriptionDefined by
  
direction
Type: String

이펙트 진행 방향을 설정합니다. 유효값은 "left","right","up", 및 "down" 입니다.

The default value is "left".		SeriesSlide
  
duration
Type: Number
이펙트가 총 재생 시간을 밀리 세컨드 단위로 지정합니다.
즉, 모든 이펙트는 duration에서 정의한 시간 안에 재생되고 종료됩니다. The default value is 500.		SeriesSlide
  
elementOffset
Type: Number
시리즈 내 각 아이템의 이펙트를 지연 시키는 시간을 밀리 세컨드 단위로 지정합니다.
elementOffset 를 0 으로 설정하면 모든 아이템의 이펙트가 동시에 시작해 동시에 끝나게 됩니다.
The default value is 20.		SeriesSlide
  
id
Type: String

id를 설정합니다. id는 사용자에 의해 정의된 임의의 고유값입니다.

설정된 id를 통해 해당 객체에 접근이 가능합니다.
SeriesSlide
  
minimumElementDuration
Type: Number

개개의 아이템의 이펙트가 완료하는데 필요한 시간을 밀리 세컨드 단위로 지정합니다.
duration 값이 이 값보다 작은 경우 duration 은 무시됩니다.

The default value is 0.		SeriesSlide
  
offset
Type: Number

이펙트의 지연 시간을 밀리 세컨드 단위로 지정합니다.
다수의 시리즈를 정의했을 때, 시간 차이를 내며 이펙트를 재생하고자 하는 경우 이 속성을 사용합니다.

The default value is 0.		SeriesSlide
